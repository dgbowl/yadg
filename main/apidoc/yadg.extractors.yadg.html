yadg: For legacy yadg files

An extractors for data in legacy yadg formats.

yadg.extractors.yadg.json module

This module can be used to update yadg json legacy datagrams.

Usage

Available since yadg-7.0.

pydantic model dgbowl_schemas.yadg.dataschema_7_0.filetype.Yadg_json
Config:

  • extra: str = forbid

Validators:

field filetype: Literal['yadg.json'] [Required]
field suffix: list[str] | None = ['.json']

Schema

xarray.DataTree:
  {{ step_name }}
    coords:
      uts:             !!float           # Unix timestamp
    data_vars:
      {{ values }}:    (uts)             # Data present in source datagram

Uncertainties

  • all values: taken as explicit from previous yadg data.
