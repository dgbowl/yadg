yadg version 5.2

Warning

This version of yadg is not yet released.

Developed in the ConCat Lab at Technische Universität Berlin (Berlin, DE).

New features in yadg-5.2 are:

  • Implemented support for merged/appended files in mpr. Test files taken from the data available at https://zenodo.org/doi/10.5281/zenodo.12165685. Thanks to Arnd Koeppe for bringing the issue up.

  • Implemented support for the Modular Potentio technique in mpr. Thanks to Graham Kimbell and Clea Burgel (from Empa) for providing test data.

Other changes in yadg-5.2 are:

Bug fixes in yadg-5.2 include:

  • Increased supported I-range values in eclab up to 193.

  • Implemented support for a 33-parameter GCPL file in mpr.
